◆ owner

template<concepts::ptr T>
using owner = T

GSL-like owning raw pointer typedef.

fl::owner<T> is designed as a safety mechanism for code that must deal directly with raw pointers that own memory. Ideally such code should be restricted to the implementation of low-level abstractions. fl::owner<T> can also be used as a stepping point in converting legacy code to use more modern RAII constructs, such as smart pointers.
