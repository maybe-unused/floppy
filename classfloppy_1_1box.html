Detailed Description

template<typename T>
class floppy::box< T >

Box memory class.

The box class behaves very similarly to the unique_ptr, but unlike the unique_ptr, it is 100% guaranteed to be filled at all times (unless the user tries to access it after calling std::move does something else that is clearly bad practice).

Reference implementation is taken from the reflect-cpp library. API reference matches (almost) the Rust Language implementation.

Template Parameters
TType of data stored in the box.
See also
https://github.com/getml/reflect-cpp/blob/main/include/rfl/Box.hpp
https://doc.rust-lang.org/std/boxed/struct.Box.html

The caller is responsible for deleting the returned pointer. Calling this function leaves the box in an invalid state.
