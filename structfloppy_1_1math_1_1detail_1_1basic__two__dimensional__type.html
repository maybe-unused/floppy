◆ lerp()

template<typename T , typename U , concepts::num N>
constexpr auto lerp ( T const & other,
T t ) const -> T
inlineconstexpr

Linearly interpolate between this basic_two_dimensional_type and another basic_two_dimensional_type.

Example:

using floppy::math::basic_two_dimensional_type;
auto const from = T(0.0F, 10.0F);
auto const to = T(8.0F, -4.0F);
fmt::println("{}", from.lerp(to, -1.0F));
fmt::println("{}", from.lerp(to, 0.0F));
fmt::println("{}", from.lerp(to, 0.5F));
fmt::println("{}", from.lerp(to, 1.0F));
fmt::println("{}", from.lerp(to, 2.0F));

Output:

[-8.0, 24.0]
[0.0, 10.0]
[4.0, 3.0]
[8.0, -4.0]
[16.0, -18.0]
Parameters
otherThe other basic_two_dimensional_type.
tInterpolation factor.
Returns
The interpolated basic_two_dimensional_type.
