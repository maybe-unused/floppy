◆ floppy::plain_type

struct floppy::plain_type
template<typename T>
struct floppy::plain_type< T >

Remove all cv qualifiers, references and pointers from a type.

E.g:
- std::string& becomes std::string;

  • const int* becomes int.
  • int** becomes int.
  • const volatile std::string** becomes std::string.
    Note
    Use plain_type_t<T>::maybe_pointer_type instead of plain_type_t<T>::type if you want to keep pointer, but remove cv and references.
    Template Parameters
    TThe type to remove qualifiers from.
    See also
    plain_type_t

Returns C++ type name or signature from typeid object using RTTI.

Example usage:

auto name = rtti::type_name<int>();
assert(name == "int");
auto name = rtti::type_name<std::vector<int>>();
assert(name == "std::vector<int>");

Generic bitwise or operator for enum types.

Requires opt-ins enable_bitwise_or or enum_flag to be defined for the enum type. Example:

enum class Example {
A = 1,
B = 2,
C = 4
};
// opt-in bitwise or
void consteval enable_bitwise_or(Example);
// alternatively, opt-in all operators
void consteval enum_flag(Example);
auto value = Example::A | Example::B;
