floppy::source_location struct

Provides information about the location of the current source file.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

Class provides information about the location of the current source file. This class is a port of C++20 std::source_location. Example usage:

int main() {
  llog::debug("I am called from {}, {}:{} in file {}!",
    lf::source_location::current().function_name(),
    lf::source_location::current().line(),
    lf::source_location::current().column(),
    lf::source_location::current().file_name()
  );
  llog::debug("I am called from {}", lf::source_location::current());
}

Output:

I am called from main, 5:7 in file main.c++!
I am called from main.c++:main 6:7

Note

This implementation can be aliased to std::source_location, if the compiler supports it. Otherwise, it is a self-contained port of C++20 std::source_location.
