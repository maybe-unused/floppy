auto floppy::panic(std::string_view message, source_location location = source_location::current()) -> void

Terminates program execution with detailed error message.

Parameters
message Error message.
location Source location of the panic. Defaults to current location.

Logs the error message and terminates program execution via std::terminate

Note

You can catch the panic via std::set_terminate custom handler, but it's not recommended.

See also

https://en.cppreference.com/w/cpp/error/terminate

https://en.cppreference.com/w/cpp/error/terminate_handler
