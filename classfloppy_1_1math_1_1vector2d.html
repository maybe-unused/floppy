Detailed Description

template<typename U = default_unit, concepts::num T = f32>
class floppy::math::vector2d< U, T >

A two-dimensional vector tagged with a unit.

Template Parameters
UAssociated unit of measurement. Default is default_unit.
TNumber type. Must satisfy concept floppy::concepts::num. Default is f32.
See also
floppy::math::length
floppy::math::size2d
floppy::math::point2d
