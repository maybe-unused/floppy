Detailed Description

template<typename U = default_unit, concepts::num T = f32>
class floppy::math::point2d< U, T >

A two-dimensional point tagged with a unit.

Template Parameters
UAssociated unit of measurement. Default is default_unit.
TNumber type. Must satisfy concept floppy::concepts::num. Default is f32.
See also
