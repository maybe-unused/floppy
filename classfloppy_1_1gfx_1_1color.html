Detailed Description

Class representing a color.

The color type is used by the graphics module to represent colors. Provides useful functions to convert between different color formats.

◆ color() [4/9]

constexpr color ( u32 rgba)
inlineexplicitconstexpr

Constructs a color from 32-bit unsigned integer.

The upper 8 bits are the red component, the next 8 bits are the green component, the next 8 bits are the blue component, and the last 8 bits are the alpha component. For example, the color #32a852 is represented as 0x32a852FF and color #10101010 is represented as 0x10101010.

Note
If passed integer is greater than 0xFFFFFF, the lower 24 bits are used as red component, the next 8 bits are used as green component, the next 8 bits are used as blue component, and the last 8 bits are used as alpha component.
Parameters
rgba32-bit unsigned integer.
